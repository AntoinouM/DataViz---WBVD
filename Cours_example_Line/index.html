DataSet

This visualization shows the weather data of Vienna. More specifically from 11.03.2020 to 10.03.2021.

The dataset contains 365 days, where each day is represented by a JSON object, which contains all information. The data is retrieved from the Dark Sky API. Unfortunately the Dark Sky API is shut down (state 2023) so we can't grab newer data.
